# Technology Stack

**Analysis Date:** 2026-03-18

## Core Technologies
- **Frontend Framework:** React 19.2.0 with Vite 7.3.1.
- **Backend Framework:** Express 5.2.1.
- **Programming Language:** TypeScript (~5.9.3).
- **Database/ORM:** PostgreSQL with Prisma 6.
- **Styling:** Tailwind CSS 4.2.1, Radix UI, Shadcn UI components.
- **State Management:** Zustand 5.0.11.
- **Form Handling:** React Hook Form 7.71.2 with Zod 4.3.6 validation.
- **Routing:** React Router 7.13.1.

## Key Dependencies
- **Client:** Axios (API), Lucide React (Icons), Recharts (Analytics), Motion (Animations), date-fns.
- **Server:** jsonwebtoken (Auth), bcryptjs (Hashing), Multer & Sharp (File/Image processing), Nodemailer (Emails), express-rate-limit/helmet (Security).
