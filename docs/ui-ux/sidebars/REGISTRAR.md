# Sidebar: REGISTRAR Role (JHS Multi-SCP Framework)

**Document Version:** 5.0.0
**Institution:** Enriqueta Montilla de Esteban Memorial High School
**Role:** `REGISTRAR`

---

## 1. Primary Operational Groups

### 1.1 Phase 1: Early Registration
- **F2F Intake:** Entry point for walk-in G7 applicants. Supports SCP selection and LRN validation.
- **SCP Funnel:** specialized dashboard for tracking all 6 program funnels (STE, SPA, SPS, SPJ, SPFL, SPTVE).

### 1.2 Phase 2: Enrollment
- **Dashboard:** Monitoring enrollment pace and the October 31 compliance countdown.
- **Official Enrollment:** processing qualifiers and transferees to `ENROLLED` status after Brigada.

### 1.3 Records & Management
- **Students (SIMS):** Full access to multi-year learner records and assessment tags.
- **Audit Logs:** Read-only access to ranking trails and status changes.
- **Sections:** Control interface for the **Autonomous Sorting Engine** (Priority 1-3 logic).

---

## 2. Dynamic Interaction Rules

- **Grade Block:** Form fields for SCP candidates automatically disable the "Next" button if Grade 6 Avg < 85.00.
- **Sorting Logic:** The "Finalize" action in the Sections view is disabled until program-capacity quotas are reviewed.
