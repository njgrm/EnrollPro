# Sidebar: SYSTEM_ADMIN Role (JHS Multi-SCP Framework)

**Document Version:** 5.0.0
**Institution:** Enriqueta Montilla de Esteban Memorial High School
**Role:** `SYSTEM_ADMIN`

---

## 1. Exclusive Governance Groups

The System Admin sidebar inherits all REGISTRAR items and adds three critical control surfaces:

### 1.1 Group: System Control
- **User Management:** Provisioning roles for SCP Coordinators, Sports Coaches, and Clinic Staff.
- **Email Logs:** auditing automated notifications dispatched to parents during the Gates.
- **System Health:** Monitoring database performance during peak July enrollment.

### 1.2 Group: Program Configuration
- **SCP Parameters:** Establishing prerequisites per program (85% threshold, medical cert requirements).
- **Ranking Logic:** configuring the weighted components (60/20/20 standard) for the ranking service.
- **Sectioning Rules:** Setting hard-caps for SCP sections and star-section quotas.

---

## 2. Compliance Oversight

- **Compliance Dashboard:** Cross-school visibility into `TEMPORARILY_ENROLLED` counts and timer statuses.
- **Audit Oversight:** specialized view for auditing ranking executions to ensure fairness and defensibility.
