# School Branding and UI Styling - Enriqueta JHS Multi-SCP

## Enriqueta Montilla de Esteban Memorial High School

**Document Version:** 5.0.0  
**Project:** EnrollPro JHS Enrollment System (Grades 7-10)

---

## 1. Branding Baseline

All public and staff-facing interfaces must use the official Enriqueta and DepEd visual identity.

Brand assets:
- Official school and DepEd logos
- School-approved primary and accent colors
- Consistent typography for policy-sensitive enrollment workflows

---

## 2. State and Badge Styling Rules

UI states must map directly to the enrollment blueprint lifecycle:
- `PRE_REGISTERED_BEC`
- `[SCP]_QUALIFIED`
- `TEMPORARILY_ENROLLED`
- `ENROLLED`

Presentation rules:
- Success styles only for qualified or finalized states.
- Warning styles for pending documentary compliance.
- Destructive styles for failed validation gates.

---

## 3. Timeline-Aware UI Context

- **Jan-May context:** Early Registration and SCP screening language
- **July-August context:** Official enrollment, Brigada encoding, and sectioning finalization

Headers and navigation labels must use **JHS Grades 7-10** wording and avoid single-program framing.
