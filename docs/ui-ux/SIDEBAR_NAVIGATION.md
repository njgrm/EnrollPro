# Sidebar Navigation: Enriqueta JHS Multi-SCP Framework

**Document Version:** 5.0.0
**Institution:** Enriqueta Montilla de Esteban Memorial High School
**Project:** EnrollPro

---

## 1. Dynamic Sidebar Context

The sidebar provides access to the multi-phase enrollment journey, filtered by role and current school phase.

### 1.1 Core Navigation Groups

| Group | Nav Items | Phase Context |
|---|---|---|
| **Early Registration** | F2F Intake, SCP Funnel | Phase 1 active (Jan-May). |
| **Enrollment** | Dashboard, Official Enrollment | Phase 2 active (July). |
| **Records** | Students (SIMS), Audit Logs | Persistent access. |
| **Configuration** | Sections, Settings, Teachers | Admin/Registrar oversight. |

---

## 2. Special Features

- **SCP Funnel Dashboard:** Central hub for Gate 2 (Assessment) and Gate 3 (Interview) score entry for STE, SPA, SPS, SPJ, SPFL, and SPTVE.
- **Autonomous Ranking:** Accessible to Registrar/Admin to execute the 60/20/20 algorithm.
- **Autonomous Sorting Engine:** Accessible via "Sections" to trigger roster generation based on JHS Blueprint priorities.

---

## 3. Interaction Rule: Compliance Alerts

- **The 85% Rule:** Real-time block in intake forms for SCP applicants.
- **Grace Period Timer:** Dashboard widget shows `TEMPORARILY_ENROLLED` learners nearing the October 31 deadline.
