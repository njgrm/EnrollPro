ENROLLPRO MASTER SYSTEM BLUEPRINT: Enriqueta Montilla de Esteban Memorial High School
For School Administrators (Principal, Registrar, Coordinators)
Date: April 10, 2026

--------------------------------------------------------------------------------
1) CORE SCOPE (JUNIOR HIGH SCHOOL: GRADES 7-10)
--------------------------------------------------------------------------------

This system manages the end-to-end enrollment journey for the Basic Education Curriculum (BEC) and all Special Curricular Programs:
- Science, Technology, and Engineering (STE)
- Special Program in the Arts (SPA)
- Special Program in Sports (SPS)
- Special Program in Journalism (SPJ)
- Special Program in Foreign Language (SPFL)
- Special Program in Technical-Vocational Education (SPTVE)

Policy Alignment: DepEd Order No. 3, s. 2018 & DepEd Memorandum No. 032, s. 2024.

--------------------------------------------------------------------------------
2) PHASE 1: EARLY REGISTRATION & DYNAMIC SCP SCREENING (JANUARY – MAY)
--------------------------------------------------------------------------------

GATE 1: PREREQUISITES
- Requires Grade 6 average (Threshold: 85.00) for all SCP selections.
- Program-specific doc checks (Medical certs, Portfolios, Sports awards).

GATE 2: THE ASSESSMENT ENGINE
- Specialists encode raw scores for program-specific tests (Written exam, Audition, Fitness, Writing, Aptitude).

GATE 3: ALGORITHMIC RANKING
- Formula: (Assessment * 60%) + (Interview * 20%) + (Q1/Q2 Avg * 20%).
- Top candidates tagged as [SCP]_QUALIFIED based on program capacity.
- Non-qualifiers automatically redirected to the PRE_REGISTERED_BEC pool.

--------------------------------------------------------------------------------
3) PHASE 2: OFFICIAL DEPED ENROLLMENT (JULY – AUGUST)
--------------------------------------------------------------------------------

VALIDATION
- INCOMING G7 / TRANSFEREES: Validate final SF9 grades and "Promoted" status. Complete full BEEF.
- BRIGADA: Mandatory capture of reading levels and BMI data in SIMS.
- RETURNING LEARNERS (G8-10): Fast-tracked using Confirmation Slip; retention grades verified for SCP continuity.

--------------------------------------------------------------------------------
4) AUTONOMOUS SORTING ENGINE (SECTIONING)
--------------------------------------------------------------------------------

Priority 1 (Hard Caps): Fills SCP sections strictly using program qualifiers.
Priority 2 (Star Sections): Fills BEC Sections 1-5 by ranking the top finalized averages.
Priority 3 (Standard): Remaining learners distributed via round-robin across Section 6+.

--------------------------------------------------------------------------------
5) COMPLIANCE & EXCEPTION PROTOCOLS
--------------------------------------------------------------------------------

- NO COLLECTION LOCK: 100% compliant with DO 19, s. 2008.
- GRACE PERIOD: TEMPORARILY_ENROLLED status for missing docs with an October 31 compliance timer.
- RECORD CONTINUITY: SIMS module automatically generates digital SF10 requests for transferees.
